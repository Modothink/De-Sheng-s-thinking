# 唯物辩证法
唯物辩证法是马克思批判吸收了恩格尔和黑格尔著作中先进的思想，从而建立的。  
《资本论》就是其运用唯物辩证法对商品社会的分析的典范。  
列宁和斯大林在资本主义的最后阶段也就是帝国主义阶段发扬了唯物辩证法。  
毛泽东在中国发扬了唯物辩证法。
